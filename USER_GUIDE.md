## Getting Started

### Logging In
1. Navigate to your CompassIQ URL
2. Enter your email and password
3. Click "Sign In"

### First-Time Setup
1. Complete the onboarding wizard
2. Configure your organization settings
3. Set up data connections
4. Customize your branding (optional)

### Interface Overview
- **Sidebar**: Main navigation on the left
- **Top Bar**: Organization switcher, demo mode, user menu
- **Main Content**: Dashboard and module content
- **Presentation Mode**: Toggle for client-facing presentations

## Command Center

The Command Center is your executive dashboard providing a real-time view of key business metrics.

### Key Features
- **KPI Cards**: Revenue, pipeline, AR outstanding, work orders
- **Alerts Panel**: Active alerts requiring attention
- **Quick Actions**: Common tasks and shortcuts

### Using the Command Center
1. Review KPI cards for current business health
2. Check alerts panel for items requiring attention
3. Click "Refresh" to update data
4. Use quick actions for common tasks

### KPIs Displayed
- **Revenue**: Current period revenue
- **Pipeline (30/60/90 days)**: Sales pipeline by timeframe
- **AR Outstanding**: Accounts receivable balance
- **Work Orders**: Active work orders
- **Active Accounts**: Total active customer accounts

## Operate Section

### Operate Dashboard
Central hub for day-to-day operations.

### Alerts
Monitor and manage system alerts.

#### Alert Types
- **Critical**: Immediate attention required
- **Warning**: Review recommended
- **Info**: Informational notifications

#### Managing Alerts
1. Navigate to **Operate > Alerts**
2. View active alerts
3. Click an alert for details
4. Acknowledge or resolve alerts
5. Set up alert rules in Settings

### Tasks
Track and manage tasks across your organization.

#### Task Features
- Task creation and assignment
- Due dates and priorities
- Status tracking
- Task categories

#### Creating a Task
1. Navigate to **Operate > Tasks**
2. Click "Create Task"
3. Enter task details
4. Assign to team member
5. Set due date and priority
6. Save

### Meeting Mode (Cadence)
Structured meeting framework for operational cadences.

#### Features
- Pre-configured meeting agendas
- Real-time data during meetings
- Action item capture
- Meeting notes

#### Using Meeting Mode
1. Navigate to **Operate > Meeting Mode**
2. Select meeting type (Daily, Weekly, Monthly)
3. Review pre-populated agenda
4. Walk through each agenda item
5. Capture action items
6. End meeting and distribute notes

## Tips & Best Practices

### Daily Workflow
1. **Morning**: Check Command Center for KPIs and alerts
2. **Throughout Day**: Update tasks and opportunities
3. **End of Day**: Review completed items, plan tomorrow

### Weekly Cadence
1. **Monday**: Review pipeline and set weekly goals
2. **Wednesday**: Check progress against goals
3. **Friday**: Review metrics and prepare for next week

### Best Practices
- **Keep Data Current**: Update records regularly
- **Use Alerts**: Set up alerts for important thresholds
- **Leverage Meeting Mode**: Use for structured team meetings
- **Document Everything**: Add notes to records
- **Review Metrics**: Regularly review KPIs for trends
